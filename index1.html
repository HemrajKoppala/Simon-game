How to play?

  • Press any key on keyboard to start the game
  • The game begins with level 1
  • The system picks a random color
  • The app flashes colors one by one
  • Example:
    Level 1 → 🔴
    Level 2 → 🔴 🟢
    Level 3 → 🔴 🟢 🔵
  • The sequence keeps growing
  • Player should clicks buttons in the same order
    how the system flashes the colors
  • Each click is stored in a game
  • checks the user clicks with system flashes in a sequence
  • After each click
  • If correct -> continue the game
  • If wrong -> Game over
  • If user completes the full sequence:
    ->Increase level
    ->Add new random color
    ->Replay full sequence
  • High score is stored
  • After game over click any key to restart the game
